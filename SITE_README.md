# Chicken Pickles Nepal — Local demo site

Simple static website to showcase three chicken pickle products. Files added:

- `index.html` — main page
- `styles.css` — styles
- `script.js` — client-side order demo logic
- `assets/` — three SVG product images

To run locally (recommended simple method):

```bash
# from repository root
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Notes:
- This is a demo static site with a client-side order form (no real payment or backend).
- I can add payment integration, admin pages, or deploy instructions (Netlify/Vercel) if you want.
